To change post order (Ascending/descending) on category page.

== Description ==
Codeprey - Post Order gives option to change the default post order on category page in Ascending or descending order. Plug and play no need of any shortcode. This Plugin is creating in your appearance->widget and drag and drop it where you want to show.It will only work on category page.

== Installation ==

### Installation

1. Add Plugin to your wordpress
2. Go to Appearance->widgets
3. Drag and Drop Post Order Widget
4. Now it will appear on category page eg: (http://example.com/category/abc).
